## A rough translation of the reference coordinate (X,Y,Z) used in each 
## cubed spherical patch to the Cartesian coordinates.
## Note: Z always increase in the radial direction w.r.t the slice.

up    : X = x, Y = y, Z = z

down  : X = y, Y = x, Z = z

left  : X = x, Y = z, Z = y

right : X = z, Y = x, Z = y

back  : X = z, Y = y, Z = x

front : X = y, Y = z, Z = x
